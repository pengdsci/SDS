STA 319 Applied Statistics

Tentative Topics


Week 1: Setting up computing tools
  • Introduction: class structure, topics, assessments, and logistics.
  • Install R, RStudio, and possibly MikTex
  • Create R Markdown document- Knit HTML, PDF, and WORD file
  • Getting started with R: basic operations, vectors, data frames (R data sets).
  • Install and load R libraries
  • GitHub repository
Week 2: Nonparametric Bootstrap Inferences
  • Review of simple random sampling (SRS) plan
  • Sampling from empirical distribution – Bootstrap sampling
  • Bootstrap confidence intervals
  • Bootstrap hypothesis tests
  • R Applications: Case study – CI and testing about population means
Week 3: Review: Correlation and Simple Linear Regression
  • Relationship between two numerical variables
  • Linear relationship: the strength of linear correlation -coefficient of correlation
  • Least square regression model – structure, assumptions, and interpretation
  • Diagnostics, R square, and interpretation
  • R Applications: Case study - simple linear regression (SLR) with R.
Week 4: Multiple Regression
  • Turn categorical predictor variables into dummy variables
  • Assumptions, Goodness-of-fit measures, and diagnostics
  • Variable selection methods
  • Summarizing output and interpreting coefficients and R square
  • R Applications: Case study - MLR with R
Week 5: Nonparametric Bootstrapping Regression Modeling
  • Bootstrapping records
  • Bootstrapping residuals
  • Bootstrap confidence intervals of regression coefficients
  • Bootstrapping test for regression coefficients
  • R Applications: Case study – Bootstrapping regression modeling
  • Mini Project #1: Project Report – Bootstrap multiple regression model
Week 6: Categorical Regression – Simple Logistic Regression
  • Practical question and model formulation
  • Structure of the model
  • Interpretation of the regression coefficients
  • R Applications: Case study – fitting logistic regression with R
Week 7: Categorical Regression – Multiple Logistic Regression
  • Models with only categorical predictor variables – Dummy variable
  • Interpretation of coefficients
  • Variable selection methods and criteria
  • R Applications: Case study - logistic regression with R
Week 8: Predictive Modeling with Logistic Regression
  • Measures of predition performance
  • Confusion matrix and predictive errors
  • Cross-validation and ROC curve
Week 9: Frequency Regression - Simple Poisson Regression
  • Practical question and model formulation
  • Model structure and interpretation
  • Regression on rates
  • R Applications: Case-study - fitting counts and rates in biology
Week 10: Dispersed Poison Regression Modeling
  • Poison regression with more than one predictor variable
  • Categorical predictor variable – dummy variables must be defined
  • Issues of overdispersion and underdispersion
  • Quasi-Poisson models
Week 11: Concepts of Time Series
  • The nature of time series data – Dependency between observations
  • The purpose of time series – forecasting
  • Autocorrelation and moving average
  • Measures of goodness-of-forecasting
  • Types and approaches to time series modeling
  • R Applications: Illustrative examples of time series using R
Week 12: Moving Average and LOESS Smoothing
  • Decompose a time series: trend, seasonality, and random error.
  • Additive and multiplicative models
  • The moving average (MA) and double MA models
  • LOESS smoothing model: fitting, forecasting, and evaluating
  • R Applications: Case study – LOESS smoothing and forecasting
Week 13: Exponential Smoothing Models
  • Single exponential smoothing models
  • Holt’s trend models
  • Holt-Winters’ seasonal models
  • Model selection – measures of accuracy.
  • R Applications: Case-study – examples of exponential smoothing.
Week 14: Preparing project presentation
  • The types of models: Linear models, GLM, and time series models (smoothing)
  • Choose one of the three projects to present to the class
  • Submit a PPT presentation by the end of Friday via D2L.
  • Each presentation allows 15 minutes: 10 minutes for presenting and 5 minutes for QA.
